📱 앱처럼 쓰기

2. 하단 더보기 (⋯) → 공유 (⬆) 누르기
3. 더보기 (∨) → [홈 화면에 추가]
📱 삼성 인터넷 (갤럭시 기본)
1. 우측 하단 더보기 (≡) 누르기
2. [전체 페이지 추가]
3. [홈 화면] 누르기
🌐 크롬 / Edge / Brave
1. 우측 상단 메뉴 [⋮] 누르기

3. [설치] → 앱처럼 작동
