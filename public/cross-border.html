🛡️ 데이터 안전

미국 서버 처리 안내 (국외이전 동의)
버전 2.3 최종 갱신 2026-05-02 Cloudflare + E2EE Stage 2 + 카카오

소라고동의 AI 친구 기능은 Anthropic, PBC (미국 캘리포니아 본사) 의 Claude API를 사용해. 너의 대화가 보안 검증된 미국 서버에서 처리되고 응답을 받아오는 구조야.

🔒 너의 데이터, 어떻게 보호되는지

Anthropic의 보안 인증

  • ✅SOC 2 Type II 인증 — 미국 보안 감사 최고 수준
  • ✅ISO 27001 정보보안 국제 표준 인증
  • ✅HIPAA 준수 가능한 인프라 (의료 데이터 처리 가능 수준)

Anthropic의 데이터 정책

  • 📅30일 후 자동 삭제 — Anthropic 서버에서 영구 제거
  • 🚫AI 학습·재활용 절대 X — Anthropic Zero Data Retention 정책상 학습·재판매·연구 등 어떤 형태로도 외부 사용 불가
  • 🔐HTTPS / TLS 1.2+ 종단간 암호화 — 전송 중 데이터 노출 X

회사(소라고동)의 데이터 정책

  • 🚫광고에 사용하지 않음 — 외부 광고 / 마케팅 / 추적 ID 일체 X
  • 🚫제3자에게 판매하지 않음 — 어떤 형태로도 외부 공유 X
  • 🚫타 사용자에게 노출되지 않음 — RLS (Row Level Security)로 본인 데이터만 본인 접근
  • 👤회사 내부 접근 1인 한정 — 관리자 본인 외 누구도 접근 불가, 모든 접근 로그 기록
  • 🔐End-to-End 암호화 (E2EE — Stage 2 활성 ✅): 너 device 에서 미리 암호화 (AES-256-GCM, 사용자 비밀번호 PBKDF2 1M iteration 키 파생) → 회사조차 평문 못 봄. AI 호출 시점에만 평문이 Anthropic 으로 전송 (Anthropic Zero Data Retention 정책으로 학습 X 처리 후 즉시 폐기).

📤 어떤 정보가 미국으로 전송되는지

전송됨
  • 너의 입력 텍스트 (체크인·일기·대화·기록)
  • 너에 대한 컨텍스트 요약 (기분·활력 데이터, 자기 인식 모델 — 익명화)
  • 시스템 프롬프트 (소라고동 캐릭터 설정)
전송되지 않음
  • 이메일·비밀번호·결제 정보
  • 사진·음악 진주의 메타데이터 (텍스트 의미만 전송 가능)
  • 다른 사용자의 데이터

🇰🇷 SNS 로그인 (카카오) — 한국 내 처리

사용자가 카카오 로그인 선택 시:

  • 수신자: 카카오 (Kakao Corp.) — 한국 회사
  • 처리 항목:
    • 이메일 (필수) — 회원 식별자
    • 닉네임 / 프로필사진 (선택) — Supabase Kakao OAuth provider 표준 scope (override 불가). 수신 후 사용·저장·DB 보유 X / 즉시 폐기 (PIPA §3-1 데이터 최소 수집)
  • 국외이전 X (한국 내 처리)
  • 회사가 받는 정보: SNS 인증 후 OAuth token 으로 위 항목 조회
  • 사용자의 SNS 비밀번호 / 친구 목록 / 메시지 / 카카오톡 채널 / 배송지 등 X 접근

카카오 로그인 사용자도 E2EE master password 별도 설정 필수 — 정신건강 자기관찰 데이터는 SNS 와 무관하게 사용자 비밀번호로 암호화.

네이버 SNS 로그인 = V5 (휴대폰 본인 인증과 함께) 도입 예정. 네이버 정책상 SNS 가입 시 별도 비밀번호 요구 X 의무 — E2EE master password 강제 흐름과 정합 위해 V5 재검토.

동의 항목

「개인정보보호법 시행령」 제18조 민감정보 처리 의무 준수, 분리 동의:

(필수) 정신건강 자기관찰 데이터를 회사가 처리하는 것에 동의합니다.
(필수) 본 데이터를 미국 (Anthropic, Supabase, Cloudflare 본사) 으로 이전하는 것에 동의합니다.
외부 analytics 도구 (Google Analytics / Facebook Pixel 등) X — 회사는 추적 픽셀·광고 ID·외부 통계 일체 사용 X. 미래 외부 analytics 도입 시 별도 동의 받음.

가입 시 또는 첫 AI 대화 시 위 동의 항목 모달에서 확인한다. 거부 시 AI 대화·분석 기능을 사용할 수 없으며, 단 체크인·일기도 Supabase 미국 서버에 저장되므로 사실상 서비스 이용 불가.

재동의 트리거

  • 정책 버전 변경 시 (currentVersion ≠ consentVersion)
  • Anthropic 데이터 정책 중대 변경 시
  • 6개월 이상 미접속 후 첫 호출 (선택, 인지 환기)
