1. 청약철회권 (전자상거래법 제17조)

사용자는 다음 기간 내 별도 사유 없이 청약철회할 수 있다:

1.1 결제 후 7일 이내 — 환불 정책

  • 0회 사용 (한 번도 안 쓴 경우) → 100% 환불
  • 1회 이상 사용 → 비례 환불 (아래 2항 공식 적용)

1.2 "사용량" 정의 (명확화)

1회 사용 = 사용자가 메인 chat / 숙고 / 마법 / 돌연변이 임시대화 등에서 메시지 1개를 전송하고 AI 응답을 받은 사례.

다음은 "사용"으로 카운트하지 않음:

  • AI 응답 실패 (네트워크 오류, API 에러)
  • 사용자가 메시지 작성 중 취소
  • 자동 생성 (챕터 마무리·일일 질문·forceAnalyze 등 백그라운드)

1.3 디지털 콘텐츠 제한 (전자상거래법 제17조 제2항)

사용자가 결제 후 즉시 사용해 콘텐츠 가치가 현저히 감소한 경우 (= AI 토큰 사용량 발생) 청약철회가 제한될 수 있다. 회사 정책상 사용량에 비례한 환불은 항상 가능하다.

2. 비례 환불 (사용량 기반)

2.1 월 정액 구독 — 비례 환불 공식

환불액 = (월 정액 × 잔여일 ÷ 30) − 사용량 차감액
  • 잔여일: 결제일부터 환불 신청일까지 경과한 일수를 30에서 뺀 값
  • 사용량 차감액: 결제 후 실제 사용한 토큰을 단가 기준으로 환산한 금액 (Anthropic 가격 그대로 차감)

현재 요금제 (부가세 10% 포함):

  • Light 8,900원/월 — 월 한도 약 7,000원 어치 ($5)
  • Premium 25,000원/월 — 월 한도 약 21,000원 어치 ($15)
예시 (Premium):
25,000원 결제, 10일째 환불 신청, 사용 토큰 4,000원 분
환불액 = 25,000 × (30 − 10) ÷ 30 − 4,000 = 16,667 − 4,000 = 약 12,667원

2.2 추가팩 — 청약철회 30일 (전자상거래법)

월 한도 도달 시 결제하는 추가팩 (Light 5,000원 / Premium 7,000원) 의 환불 정책:

  • 추가팩 결제 후 0회 사용 시 30일 이내 100% 환불
  • 1회 이상 사용 시 잔여 한도 비례 환불 (사용량 차감)
  • 사용자 명시: 추가팩은 cap 도달 시마다 계속 결제 가능 — 환불도 각 추가팩 별 독립 적용

2.3 Legacy 충전 잔액 (V4 이전 호환)

2026-04-30 V4 결제 모델 전환 이전 사용자가 보유한 legacy charge 잔액 (credit_balance_usd > 0) 은 그대로 차감하며, 환불 요청 시:

  • 잔여 charge 잔액 100% 환불 (이미 사용된 토큰 환불 X)
  • 0 도달 후 구독 모델 (2-tier) 으로 자연 전환 안내
  • 신규 charge 결제는 더 이상 불가능 (V4 부터 폐기)

2.4 처리 기한

  • 환불 신청일로부터 3-5 영업일 내 결제 수단으로 자동 환불 (포트원 자동 환불 시스템)
  • 카드 결제: 카드사 정책상 7-14일 후 카드 명세서 반영
  • 계좌이체: 영업일 기준 1-2일 내 입금

2.5 영수증 갱신 (사업자 의무)

부분 환불 시 회사는 수정 영수증을 사용자에게 발급한다 (전자상거래법 + 부가세법):

  • 자동 발행 (앱 내 [설정 → 결제 history]에서 다운로드 가능)
  • 이메일 별도 송부 (선택)
  • 환불 일자, 환불 금액, 사용량, 잔여 결제액 명시

3. 환불 불가 사유

다음의 경우 환불이 제한된다:

  1. 사용자가 약관 / 본인 의무를 위반해 서비스 정지된 경우
  2. 무료 체험 기간만 이용하고 결제 자체가 없는 경우 (환불 대상 아님)
  3. 부정 사용 (자동화 봇 / 타인 명의 결제 등)이 확인된 경우
  4. 결제 후 30일 이상 경과 후 청약철회 요청 (단 비례 환불은 가능)

4. 환불 신청 방법

포트원 자동 환불 시스템 사용 — 회사 직접 처리 X. 결제 수단으로 자동 환불.

6. 무료 환영 토큰 소진 후 자동 결제 X

무료 환영 토큰 (3,000원 ≈ $1.43) 소진 시 자동으로 유료 전환되지 않는다. 사용자가 명시적으로 결제 진행 시에만 유료 구독으로 전환된다. 무료 토큰이 남아있는 한 시간 제한 없이 자유 사용 가능하다.

또한 자동 갱신은 default OFF 다 — 매월 결제 만료 후 사용자가 명시적으로 다시 결제해야 다음 달 이용 가능. 사용자가 옵션으로 자동 갱신 ON 시에만 자동.

이는 다크 패턴 (자동결제 함정) 회피를 위한 회사 정책이다.
