# 국외이전 동의문

**용도**: 첫 AI 대화 시 사용자에게 보여줄 모달 텍스트.
**버전**: 1.0

---

## 모달 제목

🌍 미국 서버 이용 안내

## 모달 본문 (사용자에게 보여지는 톤 — 친구처럼)

소라고동의 AI 친구 기능은 **Anthropic, PBC** (미국 캘리포니아) 의 Claude API를 사용해.

너랑 나누는 대화 메시지가 미국 서버로 전송돼서 응답을 받아오는 구조야. 너 이메일·이름·결제 정보는 안 보내고, **대화 텍스트와 자기관찰 컨텍스트**만 가.

### 어떤 정보가 가는지

- 너의 입력 텍스트 (체크인 / 일기 / 대화 / 기록)
- 너에 대한 컨텍스트 요약 (기분·활력 등 추적 데이터, 사용자 패턴)
- 시스템 프롬프트 (소라고동 캐릭터 설정)

### 어떤 정보가 안 가는지

- 이메일, 비밀번호, 결제 정보
- 사진, 음악 진주의 메타데이터 (텍스트만 갈 수 있음)
- 다른 사용자의 데이터

### Anthropic 정책 (2026-04 기준)

- **데이터는 30일 후 Anthropic 서버에서 자동 삭제** (학습에 쓰지 X 정책)
- Anthropic 자세한 처리방침: https://www.anthropic.com/legal/privacy
- Anthropic은 SOC 2 Type II 인증, ISO 27001 보유

### Stage 2 (E2EE 활성 후 — 향후)

민감 필드는 너 device에서 미리 암호화돼서 저장돼. 단 AI 호출 시점엔 클라이언트가 평문으로 Anthropic에 보내야 함 (AI가 의미 처리 위해). 그 전까지 평문 접촉 X.

## 동의 항목 (분리)

[ ] **(필수)** 정신건강 자기관찰 데이터를 회사가 처리하는 것에 동의합니다.

[ ] **(필수)** 본 데이터를 미국 (Anthropic, Supabase, Vercel) 으로 이전하는 것에 동의합니다.

[ ] **(선택)** 익명화된 사용 통계를 서비스 개선 목적으로 회사가 활용하는 것에 동의합니다.

## 거부 시

- 필수 항목 거부: AI 대화 / 분석 기능을 사용할 수 없음. 체크인·일기·직접 메모만 사용 가능.
- 단 체크인·일기도 **Supabase 미국 서버**에 저장되므로, 필수 동의 거부 시 사실상 서비스 이용 불가.
- 선택 항목 거부: 그래도 다 사용 가능. 회사 통계 X.

## 코드 활용

- `state.preferences.crossBorderConsentAt` (timestamp) — 동의 시점 기록
- `state.preferences.crossBorderConsentVersion` (string) — 동의 버전 (정책 변경 시 재동의 필요)
- 첫 AI 호출 직전 체크 → 미동의 시 모달 표시 → 동의 받은 후 진행

## 재동의 트리거

- 정책 버전 변경 시
- 6개월 이상 경과 후 첫 호출 (선택, 사용자 인지 환기용)
- Anthropic 데이터 정책 변경 시 (30일 → 변경)
