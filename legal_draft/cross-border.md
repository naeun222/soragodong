## 모달 제목

🛡️ 데이터 안전 · 미국 서버 처리 안내

## 모달 본문

소라고동의 AI 친구 기능은 **Anthropic, PBC** (미국 캘리포니아 본사) 의 Claude API를 사용해. 너의 대화가 보안 검증된 미국 서버에서 처리되고 응답을 받아오는 구조야.

### 🔒 너의 데이터, 어떻게 보호되는지

**Anthropic의 보안 인증**:
- ✅ **SOC 2 Type II** 인증 — 미국 보안 감사 최고 수준
- ✅ **ISO 27001** 정보보안 국제 표준 인증
- ✅ **HIPAA 준수 가능**한 인프라 (의료 데이터 처리 가능 수준)

**Anthropic의 데이터 정책**:
- 📅 **30일 후 자동 삭제** — Anthropic 서버에서 영구 제거
- 🚫 **AI 학습에 사용되지 않음** — Anthropic 정책상 명시적으로 학습 데이터에서 제외
- 🔐 **HTTPS / TLS 1.2+ 종단간 암호화** — 전송 중 데이터 노출 X

**회사(소라고동)의 데이터 정책**:
- 🚫 **광고에 사용하지 않음** — 외부 광고 / 마케팅 / 추적 ID 일체 X
- 🚫 **제3자에게 판매하지 않음** — 어떤 형태로도 외부 공유 X
- 🚫 **타 사용자에게 노출되지 않음** — RLS (Row Level Security)로 본인 데이터만 본인 접근
- 👤 **회사 내부 접근 1인 한정** — 관리자 본인 외 누구도 접근 불가, 모든 접근 로그 기록
- 🔐 **Stage 2 활성 후**: 너 device에서 미리 암호화 (AES-256-GCM) → 회사조차 평문 못 봄

### 📤 어떤 정보가 미국으로 전송되는지

**전송됨**:
- 너의 입력 텍스트 (체크인·일기·대화·기록)
- 너에 대한 컨텍스트 요약 (기분·활력 데이터, 자기 인식 모델 — 익명화)
- 시스템 프롬프트 (소라고동 캐릭터 설정)

**전송되지 않음**:
- 이메일·비밀번호·결제 정보
- 사진·음악 진주의 메타데이터 (텍스트 의미만 전송 가능)
- 다른 사용자의 데이터

## 코드 활용

- `state.preferences.crossBorderConsentAt` (timestamp) — 동의 시점 기록
- `state.preferences.crossBorderConsentVersion` (string, '2.0') — 동의 버전
- `state.preferences.optionalAnalyticsConsent` (boolean) — 선택 통계 동의
- 첫 AI 호출 직전 체크 → 미동의 시 모달 표시 → 동의 후 진행
