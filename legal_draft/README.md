## 활성 시점

Phase C (백엔드 프록시 + 결제) 활성 시점에 사이트 / 앱에 게시.

## 주의사항 (작성 시 고려한 거)

1. **ADHD / 의료법 회피**: "ADHD 친화 자기관찰 도구" 으로 한정. "치료" / "진단" / "완화" 표현 X.
2. **다크 패턴 회피**: 자동결제, 함정 결제 표현 명확. 사용자가 가입 시 가격 / 결제 시점 / 해지 방법 명시.
3. **민감정보 처리**: 정신건강 데이터 = 민감정보 처리. 별도 동의.
4. **국외이전**: Anthropic API = 미국. 사용자 데이터 이전. 명시 동의 필수.
5. **청약철회권**: 디지털 콘텐츠 14일. 단 즉시 사용 시 제한 가능 — 명확 명시.
