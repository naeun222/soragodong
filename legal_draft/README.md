# 법무 문서 — 게시 준비 완료

ADHD 자기관찰 도구 베타 서비스용 약관·정책. 표준약관 + 의료법 회피 + 민감정보 처리 + 국외이전 동의 + 14일 환불 + 자동 환불 (포트원).

## 파일

- **[terms.md](terms.md)** — 이용약관 (월 정액 + 충전식 + 첫 7일 무료)
- **[privacy.md](privacy.md)** — 개인정보처리방침 (민감정보 별도 동의)
- **[refund.md](refund.md)** — 환불정책 (포트원 자동 환불 + 비례 환불)
- **[cross-border.md](cross-border.md)** — 국외이전 동의문 (Anthropic 미국)

## 게시 시점

**Phase C (백엔드 프록시 + 결제) 활성 시점에 사이트 / 앱에 게시.**

## 의료법 회피 핵심 wording

- "ADHD 친화적 **자기관찰 도구**" (도구 = 사용자 본인이 사용)
- "AI가 패턴 정리·**자기 인식**을 돕는다" (인식·관찰만, 진단 X)
- "**의료기기·의약품·심리치료 서비스가 아니다**" (3조 명시)
- "어떠한 분석·관찰·제안도 의학적 진단·처방·치료를 대체하지 않는다" (3조)
- 위급 상황 → 1393 / 1577-0199 / 119 안내 (5조)
- "AI 응답·분석은 **참고용이며, 의학적 조언이 아니다**" (9조)

## 다크 패턴 회피 핵심 wording

- 무료 체험 (첫 7일) 종료 시 **자동 결제 X** (refund.md 6항 명시)
- 결제 정보 등록은 **사용자 명시 동의** 후만
- 사용량 cap 명확 표시
- 환불 자동 처리 (포트원, 영업일 3-5일)

## 민감정보 처리 동의 (PIPA 시행령 18조)

- 정신건강 자기관찰 데이터 = 민감정보 가능성
- **별도 동의** 받기 — 일반 이용약관 동의와 분리
- cross-border.md 모달에서 분리된 체크박스 (정신건강 / 국외이전 / 통계 — 3개 분리)

## 국외이전 (Anthropic, Supabase, Vercel)

- 모두 미국 소재
- 첫 AI 호출 직전 cross-border.md 모달 표시
- 사용자 동의 시점 `state.preferences.crossBorderConsentAt` 기록

## 무료 / 유료 검토 흐름 (베타 단계)

**1357 (중소기업통합콜센터) 30분 무료 상담 권장**:
- ADHD wording 의료법 위반 가능성 검토
- 14일 환불 + 즉시 사용 시 비례 환불 wording 검수
- 정신건강 데이터 = 민감정보 처리 합법 여부

**유료 자문 (선택, 30-50만 원)**:
- 정식 출시 / 사용자 10명 이상 모집 시점
- 약관 전체 검수 + 분쟁 시나리오 대비

베타 단계 (지인 5-10명)는 **1357 무료 상담 + 위 초안 그대로 게시 가능**.

## 코드 통합 흐름

1. 첫 진입 시 약관 동의 모달 (terms / privacy 한 번)
2. 첫 AI 호출 직전 국외이전 동의 모달 (cross-border)
3. 결제 시작 시 환불정책 명시 (refund 링크)
4. 모든 동의 시점 `state.preferences.consentLog[]`에 기록 (audit trail)
