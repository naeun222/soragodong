# 소라고동 개인정보처리방침

**시행일**: [TBD]
**버전**: 1.0

[개인사업자 김나은] (이하 "회사")는 「개인정보보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 준수하여 사용자의 개인정보를 보호한다.

## 1. 처리 목적

| 목적 | 처리 항목 |
|---|---|
| 회원 가입·인증 | 이메일 주소 (Supabase magic link 기반) |
| AI 대화 / 분석 서비스 제공 | 사용자 입력 텍스트 (체크인·일기·대화·기록) |
| 결제 처리 | 결제 정보 (포트원 PG사 위탁) |
| 사용량 기반 청구 | 토큰 사용량, 결제 history |
| 서비스 개선 | 익명화된 사용 통계 (개인 식별 정보 제외) |
| 법적 의무 이행 | 전자상거래법상 결제·환불 기록 보존 |

## 2. 처리 항목 + 민감정보

본 서비스는 사용자의 **정신건강 관련 자기관찰 데이터**(기분·활력·수면·고민·감정 기록·자기 인식·관계 정보 등)를 처리한다. 이는 「개인정보보호법 시행령 제18조」상 **민감정보**에 해당할 수 있어, 별도 동의를 받는다.

### 별도 동의 항목

가입 시 또는 첫 AI 대화 시 다음 두 가지 별도 동의를 받는다:

1. **정신건강 자기관찰 데이터 처리 동의** — 사용자가 입력한 기분·고민·감정 기록 처리
2. **국외이전 동의** — Anthropic, PBC (미국 캘리포니아) 서버로 사용자 메시지 전송 (자세한 내용 [국외이전 동의문](cross-border.md))

각 동의는 분리되어 있으며, 사용자가 동의 거부 시 해당 기능은 사용할 수 없다 (단 그 외 기능 일부는 사용 가능).

## 3. 보유 / 이용 기간

| 항목 | 보유 기간 |
|---|---|
| 회원 데이터 (대화·체크인·진주·전략) | 회원 탈퇴 시 즉시 삭제 |
| 결제 기록 | **5년** (전자상거래법 제6조) |
| 서비스 이용 로그 | **3개월** (통신비밀보호법) |
| 부정 사용 기록 | **1년** (악용 방지) |

회원 탈퇴 시 위 법령상 보존 의무 항목 외 모든 데이터는 즉시 삭제되며, 사용자는 탈퇴 전 데이터 export를 요청할 수 있다.

## 4. 제3자 제공 / 위탁

| 수탁자 | 위탁 업무 | 처리 항목 | 위치 |
|---|---|---|---|
| **Anthropic, PBC** | AI 응답 생성 (Claude API) | 사용자 입력 텍스트 + 시스템 컨텍스트 (이름·이메일·결제 정보 X) | 미국 (San Francisco, CA) |
| **Supabase, Inc.** | 인증 + 데이터 저장 (PostgreSQL) | 이메일, 사용자 데이터 | 미국 (AWS us-east-1) |
| **포트원 (PortOne)** | 결제 처리 | 결제 정보 (카드번호 직접 보유 X, 토큰만) | 한국 |
| **Vercel, Inc.** | 백엔드 호스팅 | 사용자 요청 라우팅 | 미국 |

**국외이전 동의**: Anthropic / Supabase / Vercel은 미국 소재. 사용자가 별도 동의 시에만 데이터 전송된다 (제2조 2항 참고).

회사는 위 수탁자 외 제3자에게 사용자 데이터를 제공하지 않는다. 단 법령상 요구가 있는 경우 (수사 영장 등) 최소 범위에서 협조한다.

## 6. 데이터 보안

1. **저장 시 암호화**: Stage 2 활성 후 — 민감 필드 클라이언트 측 AES-256-GCM 암호화. 회사도 평문 접근 불가.
2. **전송 시 암호화**: HTTPS / TLS 1.2 이상 (모든 통신).
3. **접근 통제**: 회사 내부 접근은 관리자 본인 (1인 사업자)에 한정. 접근 로그 기록.
4. **인증**: Supabase magic link (비밀번호 X, 이메일 인증 토큰만).
5. **결제 정보**: 카드번호·CVC 등 회사 직접 보유 X. 포트원 토큰화만.
6. **백업**: 자동 백업 (사용자별 격리, 암호화). Supabase 내장.

## 7. 자동 수집 정보

본 서비스는 **쿠키 / 광고 ID / 추적 픽셀을 사용하지 않는다.**

분석 도구는 회사 자체 익명화 통계만 사용 (Google Analytics, Facebook Pixel 등 외부 분석 X).

## 8. 14세 미만 사용 제한

본 서비스는 **만 14세 이상**만 가입 가능하다. 14세 미만 사용자가 가입한 사실이 확인되면 즉시 데이터 삭제 후 가입 거부한다.
