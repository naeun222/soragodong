[개인사업자 김나은] (이하 "회사")는 소라고동 서비스 사용자의 개인정보 보호를 위해 다음과 같이 처리방침을 수립·공개합니다.

## 1. 처리 목적

| 목적 | 처리 항목 |
|---|---|
| 회원 가입·인증 | 이메일 주소 (Supabase magic link 기반) |
| AI 대화 / 분석 서비스 제공 | 사용자 입력 텍스트 (체크인·일기·대화·기록) |
| 사용량 기반 결제 (Phase C 활성 시) | API 호출 토큰 수, 결제 정보 (PG사 위탁) |
| 서비스 개선 | 익명화된 사용 통계 |

## 2. 처리 항목 + 민감정보

본 서비스는 사용자의 **정신건강 관련 자기관찰 데이터** (기분·활력·수면·고민·감정 기록)를 처리합니다. 이는 「개인정보보호법 시행령 제18조」상 **민감정보**에 해당할 수 있어, 별도 동의를 받습니다.

**별도 동의 항목**:
- 정신건강 자기관찰 데이터 처리 동의
- 미국(Anthropic, PBC) 서버로의 국외이전 동의

## 3. 보유 / 이용 기간

- 회원 데이터: 회원 탈퇴 시 즉시 삭제 (단 법령상 보존 의무 있는 항목 제외)
- 결제 기록: 5년 (전자상거래법)
- 서비스 이용 기록: 3개월 (통신비밀보호법)

## 4. 제3자 제공 / 위탁

| 수탁자 | 위탁 업무 | 처리 항목 | 위치 |
|---|---|---|---|
| Anthropic, PBC | AI 응답 생성 (Claude API) | 사용자 입력 텍스트 + 시스템 컨텍스트 | **미국 (San Francisco, CA)** |
| Supabase, Inc. | 인증 + 데이터 저장 | 이메일, 사용자 데이터 | **미국 (AWS us-east-1)** |
| [PG사 — TBD: 포트원 등] | 결제 처리 | 결제 정보 | 한국 |
| Vercel, Inc. (Phase C 활성 시) | 백엔드 호스팅 | 사용자 요청 라우팅 | **미국** |

**국외이전 별도 동의 필수.** 가입 시 또는 첫 AI 대화 시 명시적 동의 화면 제공.

2. 처리 정지 요구
3. 동의 철회 (탈퇴 시)
4. 손해배상 청구

## 6. 데이터 보안

1. **저장 시 암호화**: Stage 2 활성 후 — 민감 필드 클라이언트 측 AES-256-GCM 암호화. 회사도 평문 접근 불가.
2. **전송 시 암호화**: HTTPS / TLS 1.2 이상.
3. **접근 통제**: 회사 내부 접근은 최소 인원, 로깅.
4. **백업**: 자동 백업 (사용자별 격리, 암호화).
