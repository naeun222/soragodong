## 2. 처리 항목 + 민감정보

본 서비스는 사용자의 **정신건강 관련 자기관찰 데이터**(기분·활력·수면·고민·감정 기록·자기 인식·관계 정보 등)를 처리한다. 이는 「개인정보보호법 시행령 제18조」상 **민감정보**에 해당할 수 있어, 별도 동의를 받는다.

### 별도 동의 항목

가입 시 또는 첫 AI 대화 시 다음 두 가지 별도 동의를 받는다:

1. **정신건강 자기관찰 데이터 처리 동의** — 사용자가 입력한 기분·고민·감정 기록 처리
2. **국외이전 동의** — Anthropic, PBC (미국 캘리포니아) 서버로 사용자 메시지 전송 (자세한 내용 [국외이전 동의문](cross-border.md))

각 동의는 분리되어 있으며, 사용자가 동의 거부 시 해당 기능은 사용할 수 없다 (단 그 외 기능 일부는 사용 가능).

## 6. 데이터 보안

1. **저장 시 암호화 (E2EE — Stage 2 활성)**: 민감 필드 (일기·대화·진주·전략·case formulation·위기 carousel 로그 등) 는 클라이언트 측에서 사용자 비밀번호로 파생된 키(PBKDF2 1M iterations)로 AES-256-GCM 암호화 후 저장. **회사는 평문 접근 불가**. 비밀번호 분실 시 자동 백업 fallback 가능.
2. **전송 시 암호화**: HTTPS / TLS 1.2 이상 (모든 통신).
3. **접근 통제**: 회사 내부 접근은 관리자 본인 (1인 사업자)에 한정. 접근 로그 기록.
4. **인증**: Supabase OTP (이메일 코드) + 사용자 비밀번호 (E2EE 키 파생용, 회사 저장 X).
5. **결제 정보**: 카드번호·CVC 등 회사 직접 보유 X. 포트원 토큰화만.
6. **백업**: 자동 백업 (사용자별 격리, 암호화). Supabase 내장 + 클라이언트 추가 백업 옵션.
7. **AI API 호출 시 일시 노출**: Claude API 호출 시점에는 평문 메시지가 Anthropic 서버로 전송된다 (Stage 2 E2EE 의 *불가피한 예외*). Anthropic 의 **Zero Data Retention** 정책에 따라 **AI 학습·재활용 절대 X** — 학습·재판매·연구 등 *어떤 형태로도 외부 사용 불가*. 처리 후 즉시 폐기되며 30일 내 자동 삭제. 사용자는 가입 시 별도 동의 (국외이전 §3 참고).
8. **자동 백업 (E2EE 비밀번호 분실 복구용) 평문 저장 — 명시적 예외**: 회사는 E2EE 활성 사용자의 데이터에 대해서도 **분실 복구 목적의 자동 백업 (auto-backup)** 을 별도 row 에 평문으로 보관한다. 이는 사용자 비밀번호 분실 시 데이터 영구 lock-out 회피를 위한 *불가피한 예외*다. 백업 row 도 RLS 로 본인 외 접근 차단되며, 회사 내부 접근은 관리자 본인 (1인 사업자) 으로 제한된다. 사용자가 자동 백업을 거부하려면 [설정 → 자동 백업 OFF] 가능 (단 비밀번호 분실 시 복구 불가). 사용자는 본 약관 동의 시 자동 백업 평문 저장에 동의한 것으로 본다.

## 8. 연령 제한 (만 14세 이상)

### 8.1 만 14세 이상 가입 제한
본 서비스는 **만 14세 이상** 사용자만 가입·이용 가능하다.

근거:
- **PIPA §22-2**: 만 14세 미만은 법정대리인 동의 필수 — 회사는 현재 친권자 동의 폼을 운영하지 않으므로 만 14세 미만 가입 자체를 차단한다
- **「청소년보호법」 청소년유해매체물 해당 X**: 본 서비스는 자기관찰 일기·AI 친구 대화로 술·담배·도박·성인 콘텐츠·게임 등 청소년유해매체물에 해당하지 않으며, 휴대폰 본인인증 의무 대상이 아니다
- **「정보통신망법」 본인확인 의무 대상 X**: 본 서비스는 본인확인기관 운영자가 아니며, 일반 자기관찰 도구로 본인인증 의무 대상이 아니다

가입 시 **생년월일 입력 + 만 14세 이상임을 확인하는 별도 체크박스**로 PIPA §22-2 의무를 충족한다 (한국 일반 SaaS 표준).

만 14세 미만 사용자가 부정한 방법으로 가입한 사실이 확인되면 회사는 즉시 계정을 정지하고 모든 데이터를 삭제한다 (사용자/법정대리인에게 통지).

### 8.2 정신건강 정보 민감성 — 별도 동의로 처리
정신건강 자기관찰 데이터의 민감성은 **만 14세 이상 모든 사용자에게 PIPA §23 민감정보 별도 동의** (제2조 별도 동의 항목) 로 처리하며, 이는 본인인증 의무와 별개의 PIPA 보호 layer 다. 미성년자 (만 14세-18세) 도 본 별도 동의 절차로 보호된다.
