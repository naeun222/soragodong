# 소라고동 개인정보처리방침

**시행일**: [TBD]
**버전**: 1.0

---

**Naeun Lab** (대표 김나은, 사업자등록번호 261-21-02592, 이하 "회사")는 「개인정보보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 준수하여 사용자의 개인정보를 보호한다.

## 1. 처리 목적

| 목적 | 처리 항목 |
|---|---|
| 회원 가입·인증 | 이메일 주소 (Supabase magic link 기반) |
| AI 대화 / 분석 서비스 제공 | 사용자 입력 텍스트 (체크인·일기·대화·기록) |
| 결제 처리 | 결제 정보 (포트원 PG사 위탁) |
| 사용량 기반 청구 | 토큰 사용량, 결제 history |
| 서비스 개선 | 익명화된 사용 통계 (개인 식별 정보 제외) |
| 법적 의무 이행 | 전자상거래법상 결제·환불 기록 보존 |

## 2. 처리 항목 + 민감정보

본 서비스는 사용자의 **정신건강 관련 자기관찰 데이터**(기분·활력·수면·고민·감정 기록·자기 인식·관계 정보 등)를 처리한다. 이는 「개인정보보호법 시행령 제18조」상 **민감정보**에 해당할 수 있어, 별도 동의를 받는다.

### 별도 동의 항목

가입 시 또는 첫 AI 대화 시 다음 두 가지 별도 동의를 받는다:

1. **정신건강 자기관찰 데이터 처리 동의** — 사용자가 입력한 기분·고민·감정 기록 처리
2. **국외이전 동의** — Anthropic, PBC (미국 캘리포니아) 서버로 사용자 메시지 전송 (자세한 내용 [국외이전 동의문](cross-border.md))

각 동의는 분리되어 있으며, 사용자가 동의 거부 시 해당 기능은 사용할 수 없다 (단 그 외 기능 일부는 사용 가능).

## 3. 보유 / 이용 기간

| 항목 | 보유 기간 |
|---|---|
| 회원 데이터 (대화·체크인·진주·전략) | 회원 탈퇴 시 즉시 삭제 |
| 결제 기록 | **5년** (전자상거래법 제6조) |
| 서비스 이용 로그 | **3개월** (통신비밀보호법) |
| 부정 사용 기록 | **1년** (악용 방지) |

회원 탈퇴 시 위 법령상 보존 의무 항목 외 모든 데이터는 즉시 삭제되며, 사용자는 탈퇴 전 데이터 export를 요청할 수 있다.

## 4. 제3자 제공 / 위탁

| 수탁자 | 위탁 업무 | 처리 항목 | 위치 |
|---|---|---|---|
| **Anthropic, PBC** | AI 응답 생성 (Claude API) | 사용자 입력 텍스트 + 시스템 컨텍스트 (이름·이메일·결제 정보 X) | 미국 (San Francisco, CA) |
| **Supabase, Inc.** | 인증 + 데이터 저장 (PostgreSQL) | 이메일, 사용자 데이터 | 미국 (AWS us-east-1) |
| **포트원 (PortOne)** | 결제 처리 | 결제 정보 (카드번호 직접 보유 X, 토큰만) | 한국 |
| **Cloudflare, Inc.** | 백엔드 호스팅 + 정적 자원 + Edge Functions | 사용자 요청 라우팅, 정적 파일 전송 | 미국 본사 (글로벌 edge 분산) |

**국외이전 동의**: Anthropic / Supabase / Cloudflare은 미국 소재 (Cloudflare는 글로벌 edge 네트워크). 사용자가 별도 동의 시에만 데이터 전송된다 (제2조 2항 참고).

회사는 위 수탁자 외 제3자에게 사용자 데이터를 제공하지 않는다. 단 법령상 요구가 있는 경우 (수사 영장 등) 최소 범위에서 협조한다.

## 5. 사용자(정보주체) 권리 + 행사 절차

사용자는 다음 권리를 행사할 수 있다:

1. **본인 데이터 조회·수정·삭제**
2. **처리 정지 요구**
3. **동의 철회** (탈퇴 시 자동)
4. **손해배상 청구**

### 5.1 행사 방법 (3가지 채널)

**① 앱 내 직접 (1-click)**:
- 데이터 조회: 앱 내 [나 탭] / [도서관] / [설정 → 결제 history]
- 데이터 수정: 모든 항목 ✎ 버튼으로 직접 수정
- 데이터 삭제: ✕ 버튼 또는 [설정 → 회원 탈퇴] (모든 자기관찰 데이터 즉시 삭제)
- 처리 정지: [설정 → 회원 탈퇴]

**② 이메일**: [TBD — 회사 이메일]
- 본인 인증 (가입 이메일과 일치) 후 영업일 기준 3-7일 내 처리

**③ 서면**: [TBD — 회사 주소]
- 등기우편 + 신분증 사본 첨부
- 영업일 기준 7-14일 내 처리

### 5.2 행사 시 회사 의무

회사는 정보주체의 권리 행사 요구를 받은 날부터 **10일 이내**에 처리하며, 처리 결과를 사용자에게 통지한다 (개인정보보호법 제35조).

### 5.3 거부 사유

다음의 경우 권리 행사가 제한될 수 있다 (개인정보보호법 제35조 제5항):
- 다른 법률에 의한 보존 의무 (전자상거래법 결제 기록 5년 등)
- 다른 사람의 생명·신체·재산 위해 가능 시
- 진행 중인 분쟁 해결 또는 수사 협조 시

## 6. 데이터 보안

1. **저장 시 암호화**: Stage 2 활성 후 — 민감 필드 클라이언트 측 AES-256-GCM 암호화. 회사도 평문 접근 불가.
2. **전송 시 암호화**: HTTPS / TLS 1.2 이상 (모든 통신).
3. **접근 통제**: 회사 내부 접근은 관리자 본인 (1인 사업자)에 한정. 접근 로그 기록.
4. **인증**: Supabase magic link (비밀번호 X, 이메일 인증 토큰만).
5. **결제 정보**: 카드번호·CVC 등 회사 직접 보유 X. 포트원 토큰화만.
6. **백업**: 자동 백업 (사용자별 격리, 암호화). Supabase 내장.

## 7. 자동 수집 정보

본 서비스는 **쿠키 / 광고 ID / 추적 픽셀을 사용하지 않는다.**

분석 도구는 회사 자체 익명화 통계만 사용 (Google Analytics, Facebook Pixel 등 외부 분석 X).

## 8. 미성년자 보호

### 8.1 만 14세 미만 가입 제한
본 서비스는 **만 14세 이상**만 가입 가능하다. 14세 미만 사용자가 가입한 사실이 확인되면 즉시 데이터 삭제 후 가입 거부한다.

가입 시 **생년월일 입력 + 만 14세 이상임을 확인하는 별도 체크박스**를 통해 확인한다.

### 8.2 만 18세 미만 결제 제한 (전자상거래법)
**만 18세 미만 사용자가 결제하려면 법정대리인의 동의가 필수**다 (민법 제5조, 전자상거래법).

- 결제 직전 만 18세 미만 사용자에게 **법정대리인 동의 절차** 안내 (이메일 인증 또는 동의 양식)
- 동의 없이 결제한 경우 **민법 제5조 2항에 의해 취소 가능** — 사용자/법정대리인 요청 시 환불

### 8.3 미성년자 데이터 보호 강화
- 미성년자 동의 시 법정대리인 정보 함께 수집
- 미성년자 본인 또는 법정대리인이 언제든 데이터 조회·삭제 요청 가능
- 미성년자 데이터의 마케팅·통계 활용은 별도 명시 동의 시만

## 9. 개인정보 보호책임자 (CPO)

소라고동은 「개인정보보호법」 제31조에 따라 개인정보 보호책임자를 다음과 같이 지정한다:

- **성명**: 김나은
- **직위**: 대표 / 개인정보 보호책임자 (CPO)
- **연락처**: (사업장 주소 박힘 후 갱신)
- **이메일**: soragodongapp@gmail.com

CPO는 다음 업무를 수행한다:
- 개인정보보호 계획 수립·시행
- 개인정보 처리 실태 점검 + 개선
- 정보주체 권리 행사 요청 처리
- 개인정보 침해 사고 대응

### 9.1 침해 신고 채널

개인정보 침해 발생 시 회사 CPO 또는 다음 기관에 신고:
- **개인정보분쟁조정위원회** (privacy.go.kr / 1833-6972)
- **개인정보침해 신고센터** (privacy.kisa.or.kr / 118)
- **대검찰청 사이버수사과** (1301)
- **경찰청 사이버수사국** (cyberbureau.police.go.kr / 182)

### 9.2 정보보호위원회 신고

회사는 사업자등록 후 「개인정보보호법 시행령」에 따라 정보보호위원회에 개인정보처리자 등록을 진행한다.
