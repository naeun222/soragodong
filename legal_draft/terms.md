## 제2조 (정의)

1. **"서비스"**: ADHD 친화적 *자기관찰 보조 도구*. 사용자의 일기·체크인·자유 대화 기록을 기반으로 AI가 패턴 정리·자기 인식을 돕는 웹/PWA 서비스.
2. **"사용자"**: 본 약관에 동의하고 서비스를 이용하는 자.
3. **본 서비스는 의료기기·의약품·심리치료 서비스가 아니다.** ADHD 또는 기타 정신건강 상태의 진단·치료·완화·예방 목적으로 사용할 수 없으며, 본 서비스의 어떠한 분석·관찰·제안도 의학적 진단·처방·치료·**전문가 상담을 대체하지 않는다**. AI 응답은 참고용 자기관찰 보조이며 임상적 효력을 갖지 않는다.
4. **"자기관찰 도구" 의미**: 사용자가 본인 데이터를 시간에 따라 누적하며 패턴을 *가설* 형태로 살피는 도구. AI 분석은 임상 진단이 아니며, 사용자가 *반증·재해석·거부* 가능한 가설로만 사용한다.

## 제3조 (서비스 내용)

1. 일일 체크인 (활력·기분·수면 기록)
2. AI 친구 캐릭터와의 대화 (Claude API 기반)
3. 대화 챕터 자동 분류·정리 (도서관)
4. 전략·진주·깨달음 등 자기관찰 데이터 누적
5. 큰 결정 14일 숙성 도구 ("마법의 소라고동")
6. 주간/월간/분기/연간 리뷰 자동 생성
7. **End-to-End 암호화 (E2EE)** — 사용자 데이터는 사용자 비밀번호로 파생된 키로 암호화되어 저장됨. 회사는 평문 데이터에 접근할 수 없다 (Stage 2).

## 제4조 (요금)

### 4.1 무료 체험 (신규 가입)
신규 가입 사용자에게 가입 직후 **첫 달 무료 체험** 이 자동 활성화된다 (early_light 플랜, 카드 등록 X 시점). 체험 기간 동안 일반 대화·분석·리뷰·결정 도구 등 핵심 기능을 자유롭게 이용 가능하다. 체험 기간·한도는 회사 정책에 따라 변경될 수 있다.

### 4.2 유료 이용 (구독)
무료 체험 종료 시 사용자가 **명시적으로 결제한 경우에 한해** 유료 이용으로 전환된다. 자동 전환은 없다.

**구독 요금제 (월정액, 부가가치세 10% 포함)**:
- **Light** 9,900원/월 — 월 한도 약 7,000원 어치 ($5 USD)
- **Premium** 25,000원/월 — 월 한도 약 18,000원 어치 ($13 USD), Opus 깊은 대화 일일 30번 포함
- **얼리버드** 4,900원/월 — *출시 전 가입자 한정 영구 보장 가격* (출시 후 인상 X). Light 약 80% 사용량. 카드 등록 후 30일 무료 → 30일 후 첫 자동 결제 → 매월 자동 갱신.

월 한도 도달 시 사용자는 (a) **추가팩** 결제 또는 (b) 다음 결제 cycle 대기 중 선택할 수 있다.
- 추가팩 (계속 결제 가능, 1회성): Light/얼리버드 1,500원 / Premium 2,500원
- 다음 cycle 대기 = 무료. AI 호출은 차단되지만 일기/체크인/이전 데이터 조회는 가능

### 4.3 자동 갱신 (콘텐츠산업진흥법 §25 사전고지 의무 충족)
Light / Premium 은 단건 결제 (자동 갱신 X) — 매월 사용자가 명시적으로 다시 결제해야 다음 달 이용 가능. 얼리버드는 카드 등록 = 자동 갱신 *명시적 동의* 로 본다.

자동 갱신 적용 사용자에 대해:
- **갱신 7일 전** 이메일·앱 내 인박스·카카오 알림톡 (포트원 도입 후) 으로 사전 통지 — 콘텐츠산업진흥법 §25 의무 충족
- 사용자가 갱신 직전까지 자동 갱신을 OFF할 수 있음 ([설정 → 다음 갱신 해지] 1-click)
- 자동 갱신 시점, 금액, 결제 수단 명확 표시
- 미통지 자동 결제는 환불 가능

### 4.4 결제 직전 명시 (다크 패턴 회피)

6. **본 서비스는 임상 치료/진단/전문가 상담을 대체하지 않는다는 고지**

## 제5조 (사용자 의무)

1. 사용자는 본 서비스를 의료적 진단·치료 목적으로 사용하지 않는다.
2. **위급한 정신건강 위기 상황에서는 본 서비스 대신** 다음으로 즉시 연락한다:
   - 자살예방상담전화 **1393** (24시간)
   - 정신건강위기상담전화 **1577-0199** (24시간)
   - 응급실 / **119**
3. **본 서비스를 임상 치료·전문가 상담의 대체로 사용하지 않는다.** 깊은 우울·불안·자해 충동 등 임상적 증상 시 전문가 진료를 우선한다.
4. 사용자는 본인 명의의 결제 수단으로만 결제한다.
5. 사용자는 타인의 계정·데이터에 무단 접근하지 않으며, 서비스를 부정 사용 (자동화 봇 등)하지 않는다.

### 5.1 위기 상황 자동 안내 (자살예방법 §15-6 협력 권고 충족)
본 서비스는 사용자의 메시지·일기·체크인·정서 패턴 분석 중 자살·자해 신호로 해석될 수 있는 표현·패턴을 감지하면, 외부 전문기관 (1393, 1577-0199, 119) 안내 모달을 자동 표시한다. 이는 사용자 안전을 위한 *권고 안내*이며, 의료적 진단·치료 행위가 아니다. 사용자는 본 약관 동의 시 이 자동 안내 표시에 동의한 것으로 본다. 자동 안내가 표시된 시각·트리거 종류는 사용자 단말 내 기록되며 (E2EE 암호화 — 회사 접근 불가), 사용자 본인이 분쟁 시 복호화하여 활용할 수 있다.

### 5.2 연령 제한 (만 14세 이상)
본 서비스는 **만 14세 이상** 사용자만 가입·이용 가능하다 (PIPA §22-2 — 만 14세 미만은 법정대리인 동의 필수, 현재 친권자 동의 폼 미지원).

본 서비스는 **청소년유해매체물·성인 콘텐츠·게임 등에 해당하지 않는** 자기관찰 일기 도구이므로, 「청소년보호법」 또는 「정보통신망법」상 휴대폰 본인인증 의무 대상이 아니다. 가입 시 **사용자 자기 신고** (생년월일 + 만 14세 이상임을 확인하는 체크박스) 로 PIPA §22-2 의무를 충족한다.

사용자가 만 14세 미만이면서 *허위로 만 14세 이상이라 신고*하여 가입한 경우, 그로 인한 모든 책임은 사용자 본인 (및 법정대리인) 에게 있으며, 회사는 그 사실을 인지한 즉시 계정을 정지하고 모든 데이터를 삭제한다 (사용자/법정대리인에게 통지). 회사는 허위 신고로 인한 손해에 대해 책임지지 않는다.

정신건강 자기관찰 데이터의 민감성은 별도로 PIPA §23 민감정보 별도 동의 (제2조·제5조) 로 처리하며, 이는 본인인증 의무와 별개다.

## 제6조 (회사의 의무)

1. 사용자 데이터를 PIPA(개인정보보호법) 및 정보통신망법에 따라 안전하게 처리한다.
2. 사용자가 명시적으로 동의한 범위 내에서만 데이터를 처리한다.
3. **사용자 데이터를 제3자에게 판매·공유하지 않는다.**
4. AI 서비스 제공을 위해 Anthropic, PBC (미국)의 Claude API에 사용자 메시지를 전송한다는 사실을 별도 동의받는다 (자세한 내용 [국외이전 동의문] 참고).
5. **End-to-End 암호화 (E2EE) 의무**: 회사는 사용자의 일기·대화·진주·전략 등 *민감 자기관찰 데이터*를 사용자 비밀번호로 파생된 키로 암호화하여 저장한다. 회사는 평문 데이터에 접근할 수 없다. 단 (a) Anthropic API 호출 시점의 메시지 (실시간 처리, 저장 X), (b) 결제·사용량 등 *비-민감 운영 데이터* 는 평문 처리한다.
6. 회사는 본 서비스가 의학적 조언·진단·치료를 대체하지 않음을 사용자에게 명확히 고지한다 (가입 시 + 결제 시 + 상시 약관 명시).

## 제7조 (청약철회 / 환불)

사용자는 「전자상거래 등에서의 소비자보호에 관한 법률」 제17조에 따라 결제일로부터 **7일 이내**에 청약철회할 수 있다. 단 디지털 콘텐츠 (AI 토큰) 의 경우 결제 후 즉시 사용에 따라 청약철회가 제한될 수 있으며, 이 경우에도 사용량에 비례한 환불은 항상 가능하다.

자세한 절차·예외·환불 공식은 [환불정책](refund.md) 참고.

## 제8조 (서비스 중지 / 종료)

1. 회사는 사용자의 약관 위반 시 사전 통지 후 서비스 이용을 제한할 수 있다.
2. 30일 이상 미접속 사용자의 데이터는 회사 정책에 따라 처리되며, 6개월 이상 미접속 시 사용자에게 안내 후 데이터를 삭제할 수 있다 (사용자가 명시적 동의 시 보존).
3. 서비스 종료 시 [3]개월 전 사전 공지 + 사용자 데이터 export 옵션 제공.

## 제9조 (책임 제한)

1. **본 서비스에서 제공하는 AI 응답·분석은 참고용이며, 의학적·법률적·재정적 조언이 아니다.** AI 가 제공하는 traits/values/patterns/진단/case formulation 은 *가설*이며 임상 진단이 아니다.
2. **AI hallucination 위험 고지**: LLM 특성상 AI 응답에 부정확하거나 부적절한 내용이 포함될 수 있다. 사용자는 응답을 *비판적으로* 받아들이고 의심 시 전문가에게 확인한다.
3. 회사는 사용자의 자기관찰 결과로 발생한 의사결정·결과에 대해 법적 책임을 지지 않는다.
4. **사용자가 본 서비스를 임상 치료·전문가 상담의 *대체*로 사용하여 발생한 손해에 대해서는 회사는 책임을 지지 않는다.** 회사는 약관 + 가입 안내 + 결제 안내 + 위기 carousel 등 다층 고지를 통해 *대체 사용 금지*를 사용자에게 명시한다.
5. 회사의 고의 또는 중대한 과실로 인한 손해에 대해서만 회사는 책임을 진다.
6. 천재지변·전쟁·테러·통신 두절·Anthropic API 장애 등 회사의 통제를 벗어난 사유로 인한 서비스 중단은 회사의 책임이 아니다.

2. 협의 불가 시 소비자분쟁조정 위원회 (1372) 또는 한국소비자원에 조정 신청 가능.
3. 최종적으로 사용자 주소지 관할 지방법원에서 해결한다.

## 제11조 (약관 변경)

2. 사용자가 변경된 약관에 동의하지 않을 경우 이용 계약을 해지할 수 있다.
