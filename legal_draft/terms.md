## 제1조 (목적)

이 약관은 Naeun Lab (대표 김나은, 사업자등록번호 261-21-02592, 이하 "회사")가 운영하는 "소라고동" 서비스(이하 "서비스")의 이용에 관한 권리·의무·책임을 정함을 목적으로 한다.

## 제2조 (정의)

1. **"서비스"**: ADHD 친화적 자기관찰 도구. 사용자의 일기·체크인·자유 대화 기록을 기반으로 AI가 패턴 정리·자기 인식을 돕는 웹/PWA 서비스.
2. **"사용자"**: 본 약관에 동의하고 서비스를 이용하는 자.
3. **본 서비스는 의료기기·의약품·심리치료 서비스가 아니다.** ADHD 또는 기타 정신건강 상태의 진단·치료·완화·예방 목적으로 사용할 수 없으며, 본 서비스의 어떠한 분석·관찰·제안도 의학적 진단·처방·치료를 대체하지 않는다.

## 제3조 (서비스 내용)

1. 일일 체크인 (활력·기분·수면 기록)
2. AI 친구 캐릭터와의 대화 (Claude API 기반)
3. 대화 챕터 자동 분류·정리 (도서관)
4. 전략·진주·깨달음 등 자기관찰 데이터 누적
5. 큰 결정 14일 숙성 도구 ("마법의 소라고동")
6. 주간/월간/분기 리뷰 자동 생성

## 제4조 (요금)

### 4.1 무료 충전 토큰 (체험)
신규 가입 사용자에게 가입 직후 **무료 충전 토큰**을 제공한다. 시간 제한 없이 토큰 잔량 내에서 모든 기능을 자유롭게 이용 가능하다. 무료 토큰 한도는 회사 정책에 따라 변경될 수 있으며, 현재 한도는 앱 내 [설정 → 사용량] 화면에 표시된다.

### 4.2 유료 이용
무료 토큰 소진 시 사용자가 **명시적으로 결제 정보를 등록한 경우에 한해** 유료 이용으로 전환된다. 자동 전환은 없다. 무료 토큰이 남아있는 한 무료로 이용 가능하다.

### 4.3 요금제
- **월 정액제**: 월 [TBD]원, 월별 정해진 토큰 한도 내 무제한 이용. 한도 초과 시 충전 토큰에서 차감.
- **충전식**: 사용자가 미리 일정 금액을 충전 → 사용량만큼 차감. 잔액 소진 시 추가 충전 또는 사용 중단.
- **하이브리드**: 월 정액 + 한도 초과 분 자동 충전 차감. (사용자 옵션으로 활성화)

### 4.4 자동 갱신 default OFF (다크 패턴 회피)
**구독은 자동 갱신되지 않는다 (default OFF).** 사용자가 매월 결제 만료 후 명시적으로 결제 정보 입력해야 다음 달 이용 가능하다.

사용자가 명시적으로 "자동 갱신 ON" 옵션을 활성화한 경우에만 자동 갱신된다. 이 경우:
- 갱신 [3]일 전 이메일/앱 내 알림으로 사전 통지
- 사용자가 갱신 직전까지 자동 갱신을 OFF할 수 있음 (1-click)
- 자동 갱신 시점, 금액, 결제 수단 명확 표시

### 4.5 결제 직전 명시 (다크 패턴 회피)
결제 진행 직전 사용자에게 다음을 명확히 표시한다:
1. 결제 금액 + 부가세 포함 여부
2. 결제 주기 (일회성 / 월간)
3. 자동 갱신 여부 (default OFF)
4. 해지 방법 (앱 내 [설정 → 결제] 1-click)
5. 환불 정책 링크

### 4.6 가격 변경
회사는 [30]일 전 사전 공지 후 요금을 변경할 수 있다. 변경 전 결제한 사용자는 결제 만료일까지 기존 요금으로 이용한다.

### 4.7 부가가치세
요금에 부가가치세가 포함된다 (간이과세자 단계에서는 면세 가능, 일반과세자 전환 시 별도 표시).

## 제5조 (사용자 의무)

1. 사용자는 본 서비스를 의료적 진단·치료 목적으로 사용하지 않는다.
2. **위급한 정신건강 위기 상황에서는 본 서비스 대신** 다음으로 즉시 연락한다:
   - 자살예방상담전화 **1393** (24시간)
   - 정신건강위기상담전화 **1577-0199** (24시간)
   - 응급실 / **119**
3. 사용자는 본인 명의의 결제 수단으로만 결제한다.
4. 사용자는 타인의 계정·데이터에 무단 접근하지 않으며, 서비스를 부정 사용 (자동화 봇 등)하지 않는다.

## 제6조 (회사의 의무)

1. 사용자 데이터를 PIPA(개인정보보호법) 및 정보통신망법에 따라 안전하게 처리한다.
2. 사용자가 명시적으로 동의한 범위 내에서만 데이터를 처리한다.
3. **사용자 데이터를 제3자에게 판매·공유하지 않는다.**
4. AI 서비스 제공을 위해 Anthropic, PBC (미국)의 Claude API에 사용자 메시지를 전송한다는 사실을 별도 동의받는다 (자세한 내용 [국외이전 동의문] 참고).
5. 회사는 본 서비스가 의학적 조언·진단·치료를 대체하지 않음을 사용자에게 명확히 고지한다.

## 제7조 (청약철회 / 환불)

자세한 내용은 [환불정책](refund.md) 참고.

## 제8조 (서비스 중지 / 종료)

1. 회사는 사용자의 약관 위반 시 사전 통지 후 서비스 이용을 제한할 수 있다.
2. 30일 이상 미접속 사용자의 데이터는 회사 정책에 따라 처리되며, 6개월 이상 미접속 시 사용자에게 안내 후 데이터를 삭제할 수 있다 (사용자가 명시적 동의 시 보존).
3. 서비스 종료 시 [3]개월 전 사전 공지 + 사용자 데이터 export 옵션 제공.

## 제9조 (책임 제한)

1. **본 서비스에서 제공하는 AI 응답·분석은 참고용이며, 의학적·법률적·재정적 조언이 아니다.**
2. 회사는 사용자의 자기관찰 결과로 발생한 의사결정·결과에 대해 법적 책임을 지지 않는다.
3. 회사의 고의 또는 중대한 과실로 인한 손해에 대해서만 회사는 책임을 진다.
4. 천재지변·전쟁·테러·통신 두절·Anthropic API 장애 등 회사의 통제를 벗어난 사유로 인한 서비스 중단은 회사의 책임이 아니다.

## 제10조 (분쟁 해결)

1. 본 약관에 관한 분쟁은 협의로 해결한다.
2. 협의 불가 시 소비자분쟁조정 위원회 (1372) 또는 한국소비자원에 조정 신청 가능.
3. 최종적으로 사용자 주소지 관할 지방법원에서 해결한다.

## 제11조 (약관 변경)

1. 회사는 약관을 변경할 수 있으며, 변경 시 시행일 [7]일 전 앱 내 공지 또는 이메일로 사전 통지한다.
2. 사용자가 변경된 약관에 동의하지 않을 경우 이용 계약을 해지할 수 있다.
